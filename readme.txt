1.How to build MailSteward
After clone the source code from github, you need a few additonal job to build the MailSteward.
1).Please make sure you installed office and its tool - Digital Certificate for VBA Projects. Please create one certificate and we will use later.
2).After you open the solution, please switch to Signing page of property of MailSteward project. Click the button writtin with Select from Store and choose the certificate you just created.
Now you can successfully build the solution.
